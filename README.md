# Исследование маркетплейса: метрики, сегментация и A/B-тест

Комплексный продуктовый анализ быстрорастущего маркетплейса. Данные для анализа собраны заранее с помощью SQL — [скрипт сбора данных](marketplace_data_preparation.sql).

## Контекст

Маркетплейс демонстрировал рост в первой половине года, однако с сентября показатели начали падать. Задача: разобраться в причинах снижения, найти проблемные сегменты и точки роста, проверить продуктовую гипотезу через A/B-тест.

## Что сделано

- **Предобработка данных:** проверка на дубликаты, пропуски, корректность категориальных значений и временных рамок
- **Верхнеуровневый анализ:** динамика Revenue, GMV, MAU, AOV, конверсии в покупку и в заказ
- **Юнит-экономика:** ARPU, ARPPU, CAC, ROI по каналам, Payback Period
- **Сегментация и поиск инсайтов:** воронка по регионам, ОС, возрасту и сегментам покупателей; когортный retention; анализ категорий товаров и каналов привлечения
- **A/B-тест:** проверка гипотезы о влиянии бонуса на первую покупку для пользователей TikTok — дизайн, валидация рандомизации, статистические тесты

## Результаты

- Выручка росла до мая (+282% с января), затем устойчиво снижалась — основной драйвер падения: конверсия в покупку упала с 43% до 24%
- ARPPU рос на протяжении всего года — ядро платящих пользователей оставалось активным, проблема в воронке для новых
- TikTok — наименее эффективный канал: низкий retention, отрицательный ROI во второй половине года
- A/B-тест подтвердил гипотезу: бонус на первую покупку увеличил конверсию с 0,93% до 7,51% (+706%, p < 0,001); ARPU вырос значимо, AOV и ARPPU — незначимо
- Масштабирование теста требует осторожности: медианный чек в тестовой группе ниже, возможен эффект «охотников за бонусами»
